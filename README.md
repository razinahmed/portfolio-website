# Portfolio Website

A sleek, modern developer portfolio with dark mode, smooth animations, responsive design & contact form — built with HTML, CSS & JavaScript.

## Features

- Stunning dark/light mode toggle
- Smooth scroll animations (AOS library)
- Fully responsive across all devices
- Interactive project showcase section
- Skills & expertise display with progress bars
- Working contact form with validation
- Social media integration
- SEO optimized
- Fast loading & lightweight

## Tech Stack

- **Structure:** HTML5
- **Styling:** CSS3 (Custom Properties, Flexbox, Grid)
- **Interactivity:** Vanilla JavaScript
- **Animations:** AOS (Animate on Scroll)
- **Icons:** Font Awesome
- **Fonts:** Google Fonts

## Getting Started

```bash
git clone https://github.com/razinahmed/portfolio-website.git
cd portfolio-website
# Open index.html in your browser
```

## Customization

1. Edit `index.html` to update your info
2. Modify `css/style.css` for colors and themes
3. Update `js/main.js` for behavior changes
4. Replace images in the `assets/` folder

## Deployment

Deploy easily on GitHub Pages:
1. Go to Settings > Pages
2. Set source to main branch
3. Your site will be live at `https://razinahmed.github.io/portfolio-website`
