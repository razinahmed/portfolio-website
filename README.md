## 🚀 Quick Start

### Option 1: Direct (No Installation Required)
Simply open `index.html` in your browser and you're ready to go! No build tools needed.

```bash
# Clone the repository
git clone https://github.com/razinahmed/portfolio-website.git
cd portfolio-website

# Open in your browser
# macOS
open index.html

# Windows/Linux
start index.html
# or
xdg-open index.html
```

### Option 2: Using a Local Server (Recommended)
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# Node.js (http-server)
npx http-server

# Then visit http://localhost:8000
```

## 🎨 Customization Guide

### 1. **Update Your Information**
Edit `index.html`:
- Replace name and headline
- Update about section
- Add your projects
- Insert your social links

```html
<h1>Your Name Here</h1>
<p class="subtitle">Your Professional Headline</p>
```

### 2. **Customize Colors & Theme**
Edit `css/style.css`:
```css
:root {
  --primary-color: #667eea;     /* Change primary color */
  --secondary-color: #4ecdc4;   /* Change accent color */
  --dark-bg: #0a0a1a;           /* Dark mode background */
  --light-bg: #f8f9fa;          /* Light mode background */
}
```

### 3. **Modify Animations**
Edit `js/animations.js`:
- Adjust animation timing
- Change scroll trigger points
- Customize hover effects

### 4. **Replace Images**
- Add your own images to `assets/images/`
- Update image paths in `index.html`
- Optimize images for web performance

### 5. **Update Fonts**
Edit `css/style.css` to change Google Fonts:
```css
@import url('https://fonts.googleapis.com/css2?family=Your+Font:wght@400;700&display=swap');
```

### 6. **Configure Social Links**
Update links in HTML footer:
```html
<a href="https://twitter.com/yourhandle" class="social-link">
  <i class="fab fa-twitter"></i>
</a>
```

## 🌐 Deployment

### GitHub Pages (Free)
1. Push to GitHub
2. Go to **Settings > Pages**
3. Select source: **main branch**
4. Your site is live at: `https://[username].github.io/portfolio-website`

### Netlify (Recommended)
1. Connect your GitHub repo
2. Build command: (leave empty)
3. Publish directory: (leave empty, root)
4. Deploy automatically on every push
